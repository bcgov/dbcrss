This website is being retired.

Date: July 2, 2019

In response to feedback from users downloading data through the BC Data Catalogue and iMapBC, we have replaced references to this website with a shopping cart build directly into the BC Data Catalogue and iMapBC.

Why is this being done?

  • The technical underpinnings of the Data Distribution Service standalone user interface have reached their end of life cycle and continuing to support the existing interface would have meant that we could not make much-asked-for enhancements to associated components. By retiring the old system, we will be able to upgrade the new BC Data Catalogue and iMapBC components to include several bug fixes. We will also be introducing a Data Download API.

When will this happen?

  • We will be retiring the current Data Distribution Service on July 19, 2019.
  • On August 5, 2019, we will be further upgrading the system used in the BC Data Catalogue and iMapBC, removing several bugs. At this time, we will also be making available a Data Distribution API.

What action is required?

  • For instructions about using the new custom data download interface, visit BC Catalogue Instructions for Custom Data Downloads and the video Downloading data from within iMapBC.

Any question or requests for assistance can be directed to data@gov.bc.ca.

We look forward to continuing to improve your DataBC data experience.

Regards,

The DataBC Team.
